# Explain Skill

사용자가 제공한 코드나 현재 작업 중인 파일을 분석하여 다음을 설명하세요:

1. **목적** — 이 코드가 전체적으로 무엇을 하는지
2. **핵심 로직** — 중요한 흐름이나 알고리즘을 단계별로 설명
3. **주요 함수/변수** — 중요한 식별자가 어떤 역할을 하는지
4. **주의사항** — 잠재적 문제나 개선 포인트가 있으면 간단히 언급

설명은 코드를 처음 보는 개발자도 이해할 수 있도록 명확하고 간결하게 작성하세요. 전문 용어는 필요할 때만 사용하고, 사용하면 간단히 부연 설명을 추가하세요.
